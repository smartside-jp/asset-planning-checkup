# 資産形成 健康診断

NISA・iDeCo・保険・住宅ローンなど、いまの資産形成を全体像で見直すための健康診断ツール。

## 使い方

ブラウザでアクセスして、画面の指示に従って入力するだけ。データはあなたの端末（ブラウザ）にのみ保存され、サーバーには送信されません。

- スマホ・PC 両対応
- インストール不要
- ログイン不要
- 入力データは localStorage に保存（次回アクセス時に復元）

## 機能

- 入力 25 項目 + 動的生成（家族構成・子供の進路 等）
- ねじれ検出 14 ルール（学資保険・変額保険・銀行窓販投信 等の見直し提案）
- 結果ダッシュボード 7 段階：
  1. 全体像
  2. 見直しの余地
  3. 現状アセットアロケーション（ドーナツチャート）
  4. 月支出の内訳・積立目安
  5. 将来のライフイベント タイムライン
  6. 老後資金シミュレーション（4 パターン × 3 シナリオ + 年次キャッシュフロー表）
  7. もし始めるなら（NISA / iDeCo の年齢別推奨）

## 注意事項

本ツールは一般的なシミュレーションです。個別商品の推奨ではありません。具体的な商品選定や税務・法務の相談は、有資格者（IFA・税理士・社労士・FP 等）にご相談ください。
